DDoS Protection • Malformed message defense:

Support LAND, Fraggle, Smurf, Winnuke, Ping of Death, Tear Drop, TCP Error Flag and other attack defense.


• Scanning for snooping-type attack defenses:

Support ports scanning, address scanning, TRACERT control message attack, IP source routing option attack, IP timestamp option attack, IP route record option attack and other attack defense.


• Network flood attack defense:

Supports SYN Flood, SYN-ACK Flood, ACK Flood, FIN Flood, RST Flood, TCP Fragment Flood, TCP Malformed Flood, UDP Flood, UDP Malformed, UDP Fragment Flood, IP Flood, ICMP Fragment Flood, ICMP IP Flood, ICMP Fragment Flood, ICMP Fragment Flood, Other Flood and other common network layer flood attack defense; support sweep attack, pulse attack defense.


• Session layer attack defense:

Support real source SYN Flood, real source ACK Flood, TCP connection exhaustion, Sockstress, TCP null connection and other common session layer attack defense.


• UDP reflection attack defense:

Support NTP, DNS, SSDP, CLDAP, Memcached, Chargen, SNMP, WSD and other common UDP reflection amplification attack static filtering rules;
Supports dynamic generation of filtering rules to defend against new UDP reflection amplification attacks.


• TCP reflection attack defense:

Supports the creation of static filtering rules based on network layer features;
Supports dynamic generation of TCP reflection attack filtering rules.


• TCP replay attack defense:

Supports the creation of static filtering rules based on network layer characteristics;
Support dynamic generation of TCP playback attack filtering rules.


•HTTP Application Layer Attack Defense:

Supports defense against high-frequency HTTP application attacks based on behavioral analysis/HTTP CC；
Supports machine learning-based defense against low-frequency HTTP application attacks/HTTP CC；
Supports defense against slow HTTP attacks based on behavioral analysis, includingHTTP Slow Header、HTTP Slow Post、
RUDY、LOIC、HTTP Multi-Methods、HTTP Range amplification, HTTP null connections, etc.


• HTTPS application/TLS encryption application attack defense:

Supports high-frequency HTTPS/TLS encryption attack defense;
Support slow TLS incomplete session and empty connection defense.


• DNS Application Attack Defense:

Support DNS Malformed、DNS Query Flood、NXDomain Flood、DNS Reply Flood、DNS cache poisoning defense；


• SIP application attack defense:

SIP Flood/SIP Methods Flood defense is supported, including:Register Flood、Deregistration Flood、Authentication Flood，Call Flood and other attack defense;
Support for source speed limiting.


• Customized filtering rules:

Supports customization of local software and hardware filtering rules and BGP filtering rules. flowspec filtering rules to perform remote filtering, definable fields include: source/destination IP, message length, IP protocol, IP load, source/destination ports, TCP-Flag, TCP load, UDP load, ICMP load, DNS domain name, HTTP URI, HTTP User-Agent field, SIP caller field, SIP callee field, and so on.


• Geolocation Filtering:

The blocking policy supports customization, overseas to country and domestic to province.


• Co-stack defense：

Supports IPv4/IPv6 co-stack DDoS attack defense.


• Intelligent tuning of defense strategies:

Supports snapshots of attack traffic and evaluation of defense effectiveness;
Supports automatic tuning of defense policies;
Support for automatic forensics of attacks.


• Baseline Learning:

Supports dynamic traffic baseline learning with assignable learning cycles.


• Grab bag forensics:

Support automatic packet capture and custom ACL capture based on attack events,
support online analysis of packet capture files,traceability and download local analysis.
